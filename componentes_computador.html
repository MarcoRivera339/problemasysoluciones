Componentes del computador

Hardware

  • Procesador

    • Un procesador (o CPU, por sus siglas en inglés Central Processing Unit) es el componente electrónico principal de un sistema informático encargado de ejecutar instrucciones almacenadas en la memoria. Está formado por varias unidades funcionales como la unidad aritmético-lógica (ALU), que realiza operaciones matemáticas y lógicas, y la unidad de control (CU), que coordina el flujo de datos entre los diferentes componentes del sistema. Los procesadores modernos tienen múltiples núcleos (multi-core), lo que permite la ejecución simultánea de múltiples tareas, mejorando la eficiencia y el rendimiento. Además, los procesadores utilizan caché (memoria rápida) para almacenar datos de uso frecuente y acelerar el acceso a la memoria principal.
